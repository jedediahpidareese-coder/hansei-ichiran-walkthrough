# Hansei Ichiran 1870 — A Visual Walkthrough

**Live site:** https://jedediahpidareese-coder.github.io/hansei-ichiran-walkthrough/

A 33-chapter Quarto book covering the 286 daimyo domains in the *Hansei Ichiran* (藩制一覧, 1870) — the early-Meiji Japanese government's administrative survey of every domain.

This site is the companion to the authors' political-economy research on Japan's peaceful 1871 abolition of the feudal domains (*haihan chiken*) by Reese (Creighton) and Crego (Texas Tech). The walkthrough explains:

- What Hansei Ichiran is and how it was transcribed
- Ten focal-domain walkthroughs (Saga, Satsuma, Chōshū, Tosa, Kaga, Aizu–Sendai, OREP defectors, Sankyō, Restoration upgrades, the 親藩 cluster)
- The resulting dataset of 286 domains × 130 columns
- The paper's central variables (kazoku rank, Boshin alignment, the 19 deviation cases)
- Sources and citations methodology

## Features

- **33 chapters** rendered to static HTML
- **Interactive Folium map** with 286 clickable domains and bilingual EN/日本語 popups
- **5 static cartographic maps** at 250 DPI

## Source

The Quarto source files (`.qmd`) live in a separate working research repo and are not published here. This repository contains only the rendered HTML output and is rebuilt periodically from the source.
